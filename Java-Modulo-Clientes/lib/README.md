# Carpeta lib/

Esta carpeta debe contener el JDBC Driver de MySQL.

## Instrucciones

1. Descarga MySQL Connector/J desde: https://dev.mysql.com/downloads/connector/j/
2. Selecciona "Platform Independent"
3. Descarga el archivo ZIP
4. Extrae el archivo JAR
5. Copia `mysql-connector-j-8.0.33.jar` a esta carpeta
6. Renómbralo a `mysql-connector-java-8.0.33.jar` (opcional, pero recomendado)

## Verificación

Después de descargar, verifica que el archivo existe:

```bash
ls -lh lib/mysql-connector-java-8.0.33.jar
```

El archivo debe tener aproximadamente 2-3 MB.
